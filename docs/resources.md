# Resources

Resources use `authorityprompt://company/{domain}/{resource}` URIs. Available resources are profile, facts, sources, conflicts, confidence, provenance, answer-blocks, and instructions.

## Response Layer Resources

P2 adds response resources for `response-contract`, `response-schema`, `answer-policy`, `response-priorities`, `response-variants`, and `response-quality`. These resources tell AI clients how to order facts, which claims to avoid, when to show uncertainty, and which answer variant fits the user intent.
